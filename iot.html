Obstacle Avoiding Robot Car Using Arduino & Ultrasonic Sensor

Project Overview

The obstacle Avoiding robot car it can move around in an unknown environment without colliding without surrounding objects. The car would have the capacity to detect obstacle in its path based on a predetermined distance band.

Aim and purpose

The Arduino Uno is the controlling element. It uses an Ultrasonic sensor to ident ify any obstacle in front of car. When car is moving in its proper path, the sensor sense or ident ify obstacle in the path, if it finds it will send data to the controller and in turn the controller send signals to sensor to change the direct ion of the car. The micro- controller manages motors left, right, back and front depends on sensor signals.

Project Objectives

  • The obstacle avoiding robot car it can move around in an unknown environment without colliding with surrounding objects. The car would have the capacity to detect obstacle in its path based on a predetermined distance band.
  • The car detecting the obstacle the robot would change its direction to a relatively open path by making autonomous decision which is mention in the program.
  • It does not required any other external control during its operation.
  • In real-time it would measure the distance between itself from the surrounding objects.
  • It can be operate effectively in unknown environment.
  • Obstacle avoiding robots car can be used in almost in all mobile robot navigation system
  • They can be used for household work like automatic floor cleaning, vacuum cleaning.
