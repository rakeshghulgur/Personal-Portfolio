SARA THE VOICE

Project Overview

SARA the voice (voice Assistant) project it is based on android application development and provide personal assistant using voice recognization. this program includes the functions and services like Opening call list, current time, current Date, opening music player,opening browser, robat chat , camera ,video. this are the function which can executes in this project.

Aim and purpose

According to overall description in the context,the purpose of this project is develop an Android application that provides an intelligent voice assistant with the functionalites as opening call list,current time, current Date, opening music player,opening browser, robat chat , camera ,video.

Project Objectives

  • To recognize the voice of user as Input
  • Simple User Interface
  • Accepts the comand and respond to it
  • Avoid the user manual working
  • Fun to use

Tools And Technologies Used

  • Android Studio IDE
  • Visual layout Editor
  • Android Emulator
  • Gradles
