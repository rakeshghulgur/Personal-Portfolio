STUDENT INFORMATION MANAGEMENT SYSTEM

Project Overview

Information Management System is a web application which is aimed to provide an individual information of each student that are stored in database. The mai aim of this project is to develop web application which provides all the information relates to students. only Admin can manage all the information related to students only admin have the authority to insert,modify or delete the student details.

Aim and purpose

The main objective of our web application is to manage the details of students. it manages all the information about student name,standard,rollno,parent phone number,image,place. the project is totally built at administrative end,so that admin have the authority of database.so that there is no chance of fake student information.the purpose of the project is to build a application program to reduce the manual work for managing the student information.it tracks the details of student

Software requirements

  • Front-End : HTML,CSS
  • Programming Language : PHP
  • Operating System : Windows 7 and higher
  • Editor : Brackets
  • server : Xampp
  • DataBase : MySql
